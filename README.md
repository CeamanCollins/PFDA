# PFDA - Programming for Data Analytics at ATU

## Overview

This repository contains work for the *Programming for Data Analytics (PFDA)* module at ATU. It is part of the Higher Diploma in Science in Computing (Data Analytics) program. 

The repository is organized into three folders to reflect different types of work completed during the module.

---

## Contents

### 1. Assignments

This folder contains descriptions and completed assignments for the module. Each assignment demonstrates skills in Python programming, data manipulation, and data visualization to solve various analytical tasks.

### 2. My Work

This folder contains rough work, notes, and tasks completed during laboratory exercises. These tasks include exploratory work and problem-solving for key topics covered in the module.

### 3. Project

This folder contains the final project for the module. It applies techniques and concepts learned throughout the course to conduct a more in-depth analysis.

---

## How to Use This Repository

1. **Explore Folders:** Navigate to the `assignments/`, `my-work/`, and `project/` folders for the respective content.
2. **Run Code:** Follow the setup instructions in each subfolder to run the code within your Python environment.
3. **Learn Concepts:** Review the notebooks and scripts for explanations of the tasks and their solutions.
